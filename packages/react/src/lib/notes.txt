Closure scenarios

1. User clicks the "disconnect" button
- Code 1000
- No closure reason

2. Websocket sends hangup event
- Code 1000
- Server sends closure reason "EVI hung up as a result of a call to the hang_up built in tool."


3. Abnormal closure, e.g. 1006 - network issue, or server refuses/drops connection


4. Timeout
- Code 1000
- Server sends closure reason "Chat was ended because no user message was received in 30 seconds."
